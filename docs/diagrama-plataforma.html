Meta API
Insights — Campaigns
OK
Nivel campaña · agregado por período
spend · impressions · reach · clicks actions · action_values
Le pedimos a Meta los resultados totales de cada campaña en el período seleccionado: cuánto se gastó, cuántas personas vieron el anuncio, cuántos hicieron clic, etc.
Insights — Ad Sets
OK
Nivel ad set · agregado por período
adset_id · adset_name mismos campos de insights
Lo mismo que lo anterior pero desglosado por conjunto de anuncios (el nivel intermedio dentro de una campaña). Permite ver qué segmento de audiencia funcionó mejor.
Insights — Time Series
OK
Campaña × día · time_increment=1
date_start por fila para gráficos de tiempo
Los mismos datos pero con un registro por día. Esto es lo que alimenta los gráficos de líneas donde se puede ver la evolución a lo largo del tiempo.
Objectives Map
OK
GET /campaigns?fields=id,objective · para filtrar por tipo de campaña
Meta etiqueta cada campaña con un objetivo (ventas, leads, mensajes). La plataforma lee esas etiquetas para saber en qué pestaña mostrar cada campaña.
Atribución + resolución de acciones
OK
Ventana: 7d click + 1d view (igual a Ads Manager) · Resolución por MAX entre tipos coincidentes · Tipos de lead: lead, fb_pixel_lead, leadgen_grouped, contact_total, complete_registration · Tipos de conv: messaging_conversation_started_7d/1d, total_messaging_connection, first_reply
Cuando alguien hace clic en un anuncio y después compra o deja sus datos, Meta registra esa acción. Acá definimos cuánto tiempo atrás contamos ese mérito (7 días desde el clic, 1 día desde que lo vio) y nos aseguramos de leer el número correcto aunque Meta lo reporte en varios formatos distintos. Esto hace que los números coincidan con lo que muestra Ads Manager.

Análisis
🛒 E-commerce
OK
Todas las campañas · sin filtro por objetivo
ROAS global CPA Compras Ingresos Gasto CTR · CPM · Frecuencia
Pestaña para clientes que venden productos online. Muestra cuánto dinero generaron las campañas vs. cuánto se gastó (ROAS), cuántas ventas hubo y cuánto costó cada venta (CPA).
👥 Leads
OK
Filtro por objetivo LEAD · fallback inteligente a campañas con leads reales
CPL Leads Lead rate CTR · CPC Frecuencia acumulativa
Pestaña para clientes que buscan capturar datos de personas interesadas (formularios, registros). Muestra cuántos leads se generaron y cuánto costó cada uno (CPL = costo por lead).
💬 Conversaciones
OK
Filtro por objetivo MESSAGES · fallback a campañas con conversaciones reales (fix Expren)
Costo/conv Conversaciones Tasa conv CTR · CPC Frecuencia acumulativa
Pestaña para campañas cuyo objetivo es que la gente inicie una conversación (WhatsApp, Messenger, Instagram DM). Muestra cuántas conversaciones arrancaron y cuánto costó cada una.
📐 ROAS TBREIN
En desarrollo
Segmentación CON / SIN TBREIN por campaña pendiente de implementar · placeholder visible
Una métrica especial que compararía el retorno de las campañas gestionadas por TBREIN vs. las que no lo son. Aparece en pantalla pero todavía no tiene datos reales.

Pendiente
Pruebas exhaustivas pendientes antes de lanzar
Antes de usar la plataforma como referencia oficial hay que verificar que todos los cálculos (sumas, promedios, tasas) den exactamente igual que Ads Manager. Está pendiente hacer esa comparación punto a punto.

OK

Persistencia de datos
No hay
